# Changelog

All notable changes to this repo, in reverse chronological order.

## v0.6 — Voting dApp shipped

- New `dapps/voting/` — frontend for `SimpleVoting`
- Owner-only proposal creation, vote in one click
- Live progress bars and winner highlight on closed proposals
- Status badges: open / voted / closed

## v0.5 — Vault dApp shipped

- New `dapps/vault/` — frontend for `TimeLockedSavings`
- Live countdown timer (days / hours / mins / secs)
- Deposit, top up and withdraw flows
- Lookup any address to see their public lock

## v0.4 — Todo dApp shipped

- New `dapps/todo/` — frontend for `OnChainTodo`
- Add / complete / remove tasks via on-chain transactions
- Live progress bar (total / done / pending)
- Lookup someone else's public task list
- Auto-reconnect on page load + persistent disconnect

## v0.3 — TipJar dApp shipped

- New `dapps/tipjar/` — frontend for `TipJar`
- Quick-pick amounts and custom note input
- Live tips list
- Owner-only withdraw section

## v0.2 — GuestBook dApp shipped, GitHub Pages enabled

- New `dapps/guestbook/` — frontend for `GuestBook`
- Sign and read messages on chain
- GitHub Pages deployment for free hosting
- Open at `https://hellblade1488.github.io/onchain-base/dapps/guestbook/`

## v0.1 — Initial contracts deployed

- Five Solidity contracts deployed on Base mainnet
- All contracts verified on Sourcify
- Documentation in [`CONTRACTS.md`](CONTRACTS.md)

## v0.0 — 2048 game experiment

- First experiment in this repo: a small Vite-based 2048 clone
- Removed during the cleanup focused on Base contracts and dApps
